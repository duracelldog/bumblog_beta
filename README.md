## BumBlog (개인 Blog 작성 중)

#### 1. 프로그램의 특징
* React + NodeJS (express, mongodb ..)를 활용한 개인 블로그 사이트

#### 2. 사용 기술
* React Framework (with TypeScript)
* NodeJS
  * express (서버)
  * mongoose (DB)
  * redux 
  * react-router-dom
  * ETC ...

#### 3. 실행 방법.(두 가지 방법 중 택1 - package.json scripts)
* 배포파일 생성.  
  * "build"
* 개발용으로 실행.   
  * "start"
